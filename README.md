## 🎯 Objetivo

Este repositório reúne todos os exercícios e projetos desenvolvidos durante meus estudos de HTML e CSS no Curso em Vídeo. O objetivo foi aprender os fundamentos do desenvolvimento web e praticar a criação de páginas utilizando apenas HTML e CSS.

---

## 📖 Conteúdo estudado

Durante o curso pratiquei:

* Estrutura de páginas HTML
* Tags semânticas
* Listas e tabelas
* Links e imagens
* Formulários
* CSS básico
* Cores e tipografia
* Box Model
* Flexbox
* Responsividade básica
* Organização de projetos

---

## 🛠 Tecnologias utilizadas

* HTML5
* CSS3
