principais formatações

negrito/destaque

nessa frase temos um termo em negrito usando a tag <b>(nao semantica)

Nesta frase temos um termo em desataque usando a tag <strong>(semântica)

Itálico/enfase

Nesta frase temos um termo em italico usando a tag <i>(nao semântica)

nesta frase temos um termo em ênfase usando a tag <em>

Texto marcado

Podemos criar também um texto marcadousando a tag <mark>

e no outro parágrafo, temos outro texto marcado no final

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo

Texto Deletado

podemos marcar um texto como excluído com a tag <del> para indicar que ele deve ser lido, mas não considerado

Texto Inserido

Podemos marcar um texto como inserido com a tag <ins> para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag <u>(não semantica)

Texto sobrescrito

Para inserir do tipo x20+3 basta usar a tag <sup>

Texto subscrito

Para inserir coisas do tipo H20 basta usar a tag <sub>
